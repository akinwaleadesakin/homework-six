# Homework Six

This repository contains the HTML, CSS, and SCSS files for the homework six project. It includes multiple pages representing different countries, as well as styling and assets. 

https://in-info-web4.luddy.indianapolis.iu.edu/~aadesaki/homework-seven/

## Structure

- HTML files for each page (index, about, blog, contact, tours, etc.)
- `css/` directory with compiled styles
- `scss/` directory with source Sass files
- `images/` directory for assets

## Usage

Open the `index.html` file in a browser to view the site. Styles are managed via the SCSS files and compiled into the `css` folder.
